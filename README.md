# Salary Prediction API (Node.js + Express) - Simple Guide

Train model in Python. Call Python from Node/Express to predict. Return JSON.

---

## Folder Structure

```
ML Uni/
  api/                           # Node.js + Express
    src/
      routes/
        predictions.ts           # Express routes
      controllers/
        predictionController.ts  # Request handler
      utils/

    package.json
```

## API Contract

### Request

`POST /api/v1/predictions/salary`

```json
{
  "age": 29,
  "gender": "Male",
  "educationLevel": "Bachelor's",
  "jobTitle": "Data Analyst",
  "yearsOfExperience": 4.0
}
```

### Response

```json
{
  "salary": 73450.21,
  "modelVersion": "1.0.0",
  "traceId": "7e4f2e36-4b2f-49db-95bf-9ce3f8456dcb"
}
```

### Error Response

```json
{
  "code": "VALIDATION_ERROR",
  "message": "Invalid request body",
  "details": [
    {
      "field": "educationLevel",
      "issue": "Unsupported value"
    }
  ],
  "traceId": "7e4f2e36-4b2f-49db-95bf-9ce3f8456dcb"
}
```

---

## Step 1: Python Training (`ml/train.py`)

Train once. Saves all artifacts to `model_artifacts/`.

Key points:
- Split train/test first
- Fit encoders + scaler on train set only
- Save all preprocessing mappings as JSON
- Train GradientBoostingRegressor

---

## Step 2: Python Predict Script (`ml/predict.py`)

Standalone script that Node calls via subprocess.

Takes features as command-line args:
```bash
python ml/predict.py 29 Male "Bachelor's" "Data Analyst" 4
```

Returns JSON:
```json
{
  "salary": 73450.21,
  "modelVersion": "1.0.0"
}
```

---

## Step 3: Node/Express Setup

```bash
npm init -y
npm install express zod
npm install -D typescript @types/express @types/node ts-node
```

---

## Step 4: Express App

Structure:
1. `routes/predictions.ts` - Express routes
2. `controllers/predictionController.ts` - Handle requests
3. `utils/validation.ts` - Zod validation
4. `utils/subprocess.ts` - Spawn Python
5. `main.ts` - Start server

---

## Testing Locally

1. Train model: `python ml/train.py`
2. Test Python script: `python ml/predict.py 29 Male "Bachelor's" "Data Analyst" 4`
3. Start Node: `npm start`
4. Call endpoint:
   ```bash
   curl -X POST http://localhost:3000/api/v1/predictions/salary \
     -H "Content-Type: application/json" \
     -d '{"age":29,"gender":"Male","educationLevel":"Bachelor'\''s","jobTitle":"Data Analyst","yearsOfExperience":4}'
   ```
